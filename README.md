# Image to MIDI Sonifier

このツールは、画像（PNG）を解析し、その輝度や色彩の情報を MIDI メロディとして可聴化（Sonification）するコマンドラインツールです。

## 特徴

- **輝度 (Luminance) → 音の高さ (Pitch)**: 画像の明るさを 36 (C2) から 96 (C7) のノート番号にマッピングします。
- **彩度 (Saturation) → 強弱 (Velocity)**: 色の鮮やかさを音の強さに変換します。
- **X軸スキャン**: 画像を左から右へ読み取り、時間の経過とともにメロディを生成します。
